# Modelagem — estrutura por domínio

Monorepo com módulos de **Crédito**, **Mercado**, **Liquidez**, **Juros** e **Econometria**.

## Layout
- \src/modelagem/\: pacote Python com submódulos por domínio
- \
otebooks/\: exemplos e EDA por domínio
- \docs/\: documentação e referências
- \	ests/\: testes (pytest)
- \data/\: dados brutos/tratados (não versionar conteúdo sensível)

## Instalação (dev)
\\\powershell
python -m venv .venv
. .venv/Scripts/Activate.ps1
pip install -r requirements.txt
pip install -e .
pytest
